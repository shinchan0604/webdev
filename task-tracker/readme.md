# Task Manager

A simple, front-end only To-Do list web application built using clean, vanilla JavaScript, HTML5, and CSS3.

## Features

* Add Tasks: Type a task into the field and click the button or hit enter to add it dynamically.
* Complete Tasks: Click anywhere on a task item to toggle a crossed-out completion style.
* Delete Tasks: Click the dedicated delete button on any task item to remove it from the page instantly without a reload.
* Responsive Interface: The container centered layout adjusts cleanly between standard desktop browsers and mobile devices.

## Tech Stack

* HTML5: Structural blueprint markup
* CSS3: Layout alignment using Flexbox and state styling for task elements
* JavaScript (ES6): Event handling, dynamic DOM element creation, and item destruction

## File Structure

* `index.html` - The webpage structure
* `style.css` - Component layouts and typography
* `script.js` - Dynamic interaction logic

## How to Run

1. Place `index.html`, `style.css`, and `script.js` in the exact same folder directory.
2. Open `index.html` inside any standard web browser.
